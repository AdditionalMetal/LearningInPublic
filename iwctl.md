# iwctl

Basics in setting up iwctl for future reference...

## Quick-Start - Connect to Network

```
$ iwctl

[iwd]# help

[iwd]# device list

[iwd]# device <name> set-property Powered On

[iwd]# station <name> scan

[iwd]# station <name> get-networks

[iwd]# station <name> connect <SSID>

[iwd]# 

```
